# 🌫️ AQI Discord Bot

แจ้งเตือนคุณภาพอากาศผ่าน Discord อัตโนมัติทุกเช้า 07:00 น.  
พัฒนาสำหรับพื้นที่กาฬสินธุ์ บอกได้เลยว่าวันนี้วิ่งออกกำลังกายได้ไหม

## ทำไมถึงสร้าง
อยากรู้ว่าอากาศวันนี้ดีพอจะออกกำลังกายกลางแจ้งไหม  
แทนที่จะต้องเปิดเว็บเช็คเองทุกวัน ให้ Bot ส่งมาให้เลย

## ฟีเจอร์
- ดึงค่า AQI real-time จาก IQAir API
- แปลงค่า AQI เป็นคำแนะนำภาษาไทย
- ส่งรายงานผ่าน Discord Webhook อัตโนมัติ
- แจ้งเตือนทุกวันตอน 07:00 น.

## เทคโนโลยีที่ใช้
- Python
- IQAir API
- Discord Webhook
- schedule

## วิธีติดตั้ง
1. ติดตั้ง library ที่ต้องใช้
   pip install requests schedule
2. เปิดไฟล์แล้วใส่ค่าของตัวเอง
   WEBHOOK_URL = "Discord webhook ของคุณ"
   IQAIR_TOKEN = "API key จาก IQAir"
3. รันโปรแกรม
   python AQI.py
